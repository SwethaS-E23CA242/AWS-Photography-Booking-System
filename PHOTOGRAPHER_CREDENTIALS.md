## How to Use

1. Go to: http://127.0.0.1:5000/login
2. Select "Photographer Login"
3. Enter any photographer's username and password from the table above
4. You'll be redirected to the photographer dashboard
5. All bookings for that specific photographer will be displayed
6. Only that photographer's bookings are visible - photographers cannot see other photographers' bookings

## Features Available to Photographers

- ✅ View their own bookings
- ✅ See booking dates, times, locations, and customer notes
- ✅ View booking status (Pending/Confirmed/Cancelled)
- ✅ Filter by photographer ID (each photographer only sees their own bookings)
- ✅ Logout functionality
